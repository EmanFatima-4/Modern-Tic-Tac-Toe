# Modern Tic Tac Toe

A modern and responsive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features

* Modern Glassmorphism UI
* Responsive Design
* Two Player Gameplay
* Winner Detection
* Draw Detection
* Restart Game Functionality
* Smooth Hover Animations

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```text
modern-tic-tac-toe/
│── index.html
│── style.css
│── script.js
│── README.md
└── screenshots/
    └── preview.png
```

## Installation

1. Clone the repository:

```bash
git clone https://github.com/EmanFatima-4/Modern-Tic-Tac-Toe
```

2. Open the project folder.

3. Run `index.html` in your browser.
